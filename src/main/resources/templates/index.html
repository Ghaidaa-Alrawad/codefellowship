CodeFellowship App Brief :

  • - User Authentication: Secure login with BCrypt-hashed passwords, and easy access to signup page from the login page.
  • - User Profile: Profiles include username, first and last name, date of birth, and bio, these fields set during signup and non-editable afterward.
  • - User Registration: Seamless signup process with automatic login.
  • - Homepage: Login/signup links for non-logged-in users, logout for logged-in users, first name displayed prominently for logged-in users.

If you want to see a specific user info you can visit http://localhost:8080/users/{id}, and you can put the user id that you want to view
